Juego de dados - Java Swing

Proyecto personal en Java Swing basado en el juego de dados de la película 'Piratas del Caribe'. El juego es un player vs máquina, donde se aplicó lógica pura (sin base de datos) para que la máquina (Davy Jones) responda a los patrones de juego planteados. En el proyecto también se aplico librerias de audio de java para controlar la música del juego.

-------------------------
Backend:

Java 17 - Java Swing - Librearias de audio: AudioSystem - Clip - FloatControl
