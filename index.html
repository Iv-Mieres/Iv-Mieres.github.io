Pasado

Hace diecinueve años comencé mi carrera como profesor de música práctica y teórica, la cual me terminó de formar como una persona comprometida, responsable y comunicacionalmente asertiva. Hace dos años decidí entrar a este mundo de la programación por recomendación de una amiga y la verdad es que quedé fascinado al punto de querer reconvertirme laboralmente. Comencé estudiando como autodidacta viendo videos, documentación, haciendo cursos y participando en bootcamps con grupos de trabajo para mejorar mis habilidades y poder aplicar metodología ágil.

Presente

Actualmente estoy en el desarrollo web de forma freelance, junto con un grupo de trabajo, para diferentes clientes. Además me sigo capacitando aprendiendo sobre microservicios, testing automatizado y tratando de mejorar los conocimientos que he adquirido a lo largo de estos dos años de estudio.

Futuro

Mis objetivos a futuro son mejorar mi ingles, que es uno de los puntos flojos en mi stack y seguir mejorando mis habilidades técnicas y blandas desarrollado nuevos proyectos junto con mi grupo de compañeros/as. También me gustaria conocer nuevas personas con las cuales seguir generando cierta retroalimentación en cuanto a conocimiento, y como punto final poder conseguir mi primero empleo IT dentro de una empresa, siendo este el objetivo principal por el cual nos preparamos todos en este rubro.

API Rest - Red social

El usuario registrado podrá selecionar el perfil que mejor se adapte a sus necesidades y contará con un filtro de busqueda que facilitará la comunicación entre ambas partes.

Java 17 - Spring Boot 2 - MySql - Spring Security (basic auth) - JPA/Hibernate - Swagger - lambda - API Stream

Juego de dados - Java Swing

Proyecto personal en Java Swing basado en el juego de dados de la película 'Piratas del Caribe'. El juego es un player vs environment (PvE), donde se aplicó lógica (sin base de datos) para que la máquina responda a los patrones de juego planteados. En el proyecto también se aplico librerias de audio de java para controlar la música del juego.

Backend:

Java 17 - Java Swing - Librearias de audio: AudioSystem - Clip - FloatControl
